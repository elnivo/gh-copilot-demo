# Github copilot documentation
This is the documentation for the Github copilot extension. It is a tool that helps you write code faster by providing suggestions based on the context of your code. It uses machine learning to understand the code you are writing and provides suggestions that are relevant to your code.
## Features
- Provides suggestions based on the context of your code
- Supports multiple programming languages
- Can be used in any code editor that supports extensions
## Installation
To install the Github copilot extension, follow these steps:
1. Open your code editor and go to the extensions marketplace
2. Search for "Github copilot" and click on the install button
3. Once the installation is complete, you can start using the extension by opening a code file and typing your code. The extension will provide suggestions as you type.
## Usage
To use the Github copilot extension, simply start typing your code in a code file. The extension will provide suggestions based on the context of your code. You can accept a suggestion by pressing the Tab key or by clicking on the suggestion. You can also see more suggestions by pressing the Ctrl + Space keys.
